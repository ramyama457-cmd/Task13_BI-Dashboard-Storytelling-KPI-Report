# Task 13: BI Dashboard Storytelling – KPI Report

## Objective
The objective of this task is to design and develop an executive-ready Business Intelligence dashboard that clearly communicates key performance indicators (KPIs) and business insights. The dashboard focuses on sales performance, profitability, trends, and regional analysis.

## Tools Used
- Primary Tool: Power BI Desktop  
- Alternative Tool: Tableau Public  

## Dataset
- Global Superstore Dataset  
This dataset contains retail sales data including sales, profit, product categories, regions, order dates, and product details.

## KPIs Created
- Total Sales – Sum of all sales values  
- Total Profit – Sum of all profit values  
- Profit Margin (%) – Total Profit divided by Total Sales  

## Dashboard Components

### KPI Cards
- Total Sales  
- Total Profit  
- Profit Margin (%)  

### Sales Trend Analysis
- Line chart showing total sales over time to identify growth and seasonality.

### Category Performance
- Bar chart displaying sales by product category for comparison.

### Regional Performance
- Bar chart or map visual representing sales performance by region.

### Top 10 Products
- Table showing top 10 products based on sales, including sales and profit.

### Interactive Slicers
- Region  
- Order Date  
- Category  

### Insights Section
- Text box summarizing three key business insights derived from the dashboard.

## Key Insights
- Sales show a consistent upward trend over time.  
- The West region contributes the highest share of total sales.  
- The Technology category generates the highest profit margin.

## Files Included
- dashboard.pbix – Power BI dashboard file  
- dashboard_export.pdf – Exported dashboard in PDF format  
- insights_task13.txt – Business insights document  
- README.md – Project documentation  

## How to Run the Project
1. Download or clone the repository.  
2. Open the dashboard.pbix file using Power BI Desktop.  
3. Use slicers to interact with the dashboard.  
4. Refer to the PDF for a static version of the dashboard.
